# Space Invaders

A classic Space Invaders game built with Next.js, TypeScript, and Canvas.

## Getting Started

```bash
npm install
npm run dev
```

## Controls

- **Arrow Left / A**: Move left
- **Arrow Right / D**: Move right
- **Space**: Shoot
- **Enter**: Start / Restart game

## Features

- Player ship with keyboard controls
- Alien formation that moves and descends
- Player and enemy shooting
- Score system and lives
- Progressive difficulty levels
- Retro arcade visual style
- Smooth canvas animations
